LTposter 0.3

* 다른 동작을 수행하기 전에 ISBN의 적합성을 검사합니다.
  ISBN-10과 ISBN-13의 체크섬을 검사할 수 있습니다.

* 다른 책과 엮지 않고도 책을 등록할 수 있습니다.

* 종종 책이 잘못 엮이던 문제에 대한 임시 회피 방법을
  넣었습니다.

* 출판년도 필드를 자동으로 채워넣습니다.

* 국립중앙도서관을 먼저 검색하도록 도서관 검색 순서를
  조정하였습니다.

* 데모 HTML을 소스 수정에 맞게 업데이트하였습니다.

* 파이썬 파일의 소스 코드 인코딩이 선언되지 않아 아파치
  로그에 에러가 쌓이던 문제를 수정하였습니다.

* 원서명을 찾지 못했을 때 발생하던 UnboundLocalError를
  수정하였습니다.

* 원서명이나 원저자명에 ASCII 밖의 문자가 있을 때 발생하던
  UnicodeEncodeError를 수정하였습니다.

* 에러가 발생했을 때 종종 이전 페이지로 돌아가는 링크가
  나타나지 않던 문제를 수정하였습니다.

* 라이선스를 MIT 라이선스로 결정하였습니다.

* 장석문님께서 데비안 패키징을 공헌해 주셨습니다.

LTposter 0.2

* 기존에는 국회도서관만 검색하였는데, 여러 도서관을 차례대로
  검색할 수 있도록 기능을 추가하였습니다.

  기본으로 국회도서관과 국립중앙도서관을 검색합니다. 도서관을
  추가하려면 zinfo.py 파일을 편집하면 됩니다.

* ISBN의 언어 접두사를 읽어 언어 필드를 자동으로 채워넣습니다.

* HTTP 요청을 보낼 때 User-Agent 헤더를 LTposter/0.2와 같이
  채워넣습니다.

* LibraryThing 계정 설정이 안 되었거나 잘못되어 로그인이
  실패했을 때 에러 처리를 개선하였습니다.

* 퍼미션 오류로 records 디렉토리를 삭제할 수 없던 문제를
  수정하였습니다.

* 알라딘 표지 이미지의 파일 이름 붙이는 방식이 바뀌어 최근
  추가된 표지 이미지를 가져올 수 없던 문제를 수정하였습니다.

* 표지 이미지를 찾지 못했을 때 에러 처리를 개선하였습니다.

* 국립중앙도서관 레코드 중 원저자명이 100a가 아닌 507a에 있는
  경우도 처리하도록 수정하였습니다.

* 원서명에 관사가 있는 경우 관사를 괄호로 싸게 되어 있어
  이를 풀어줘야 하는데, 기존의 A와 The 외에도 Il, La, L' 등을
  처리합니다.

  프랑스어 등에서 어포스트로피로 끝나는 관사의 경우 관사 뒤에
  빈 칸을 넣지 않고 올바르게 처리합니다.

* 원서명에 & 와 같은 기호가 들어간 경우 제외하고 검색합니다.
